Модель транспортної мережі міста з використанням бібліотеки NetworkX.

## Завдання 1
- Створено та візуалізовано граф
- Проведено аналіз кількості вершин, ребер та ступенів

## Завдання 2
- Реалізовано BFS та DFS
- Порівняно результати алгоритмів

## Завдання 3
- Додано ваги до ребер
- Реалізовано алгоритм Дейкстри

## Висновки
BFS знаходить найкоротший шлях за кількістю переходів, DFS залежить від структури графа. 
Алгоритм Дейкстри дозволяє враховувати реальні витрати на переміщення.
